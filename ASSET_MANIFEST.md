# Манифест графики

Все рисунки в `assets/figures/` созданы как авторские технические схемы для статьи `rzd_final_tm_v5_with_figures.md` / `README.md`.

Внешние фотографии и изображения из интернета в архив не включались, чтобы не тянуть в GitHub чужие права. Исходная инфографика из чата использована только как смысловой ориентир; `fig01_network_map` — новая схематическая реконструкция по числам, указанным в статье.

## Файлы

1. `fig01_network_map.svg/png` — схема сети ВСМ от Москвы.
2. `fig02_ballastless_track.svg/png` — поперечник безбалластного пути на слабом грунте.
3. `fig03_wheel_rail_contact.svg/png` — пятно контакта «колесо — рельс».
4. `fig04_700t_beam_scale.svg/png` — балка 700 т в масштабе с автобусом и вагоном.
5. `fig05_pantograph_catenary.svg/png` — пантограф, контактная сеть, волна в проводе.
6. `fig06_aluminum_profile_980mm.svg/png` — кузовной алюминиевый профиль 980 мм и зона сварки.
7. `fig07_braking_curve.svg/png` — тормозная кривая и точка безопасной остановки.
8. `fig08_world_comparison.svg/png` — сравнение скоростей «Сапсан», Shinkansen, TGV, CR400, ЭВС360 и Maglev.

## Как использовать

- GitHub хорошо отображает SVG прямо в Markdown.
- PNG-дубликаты лежат рядом на случай публикации на площадках, где SVG не принимается.
- Основной файл для пуша: `README.md`.
